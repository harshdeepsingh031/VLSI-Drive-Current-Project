# VLSI Drive Current Project

## Introduction
This project compares FinFET and MOSFET technologies based on their electrical performance.

## Features
- Comparison of drain current
- Graphical visualization using Matplotlib
- Analysis of scaling effects

## Tools Used
- Python
- NumPy
- Matplotlib

1. Install Python  
2. Install required libraries:
   pip install numpy matplotlib

## Results
FinFET shows better performance in terms of higher drive current and reduced leakage compared to MOSFET.
